# 🚀 DFS (Depth-First Search) 알고리즘 구현

Python의 **재귀(Recursion)**를 사용하여 깊게 먼저 탐색하는 **깊이 우선 탐색(DFS)**의 구현 방법을 기록한다.

---

## 1. 개요
**DFS**는 그래프 탐색 알고리즘으로, 현재 노드와 연결된 가지를 따라 최대한 깊이 내려갔다가, 더 이상 갈 곳이 없으면 다시 돌아와 다른 가지를 탐색하는 방식이다.

* **주요 용도:** 모든 노드 방문, 경로의 특징을 저장해야 하는 문제, 사이클 검출 등
* **자료구조:** 재귀 함수(내부적으로 시스템 스택 사용)

---

## 2. 구현 코드 (Python)



```python
# 탐색할 그래프 (인접 리스트 방식)
graph = {
    1: [2, 3, 4],
    2: [1, 5],
    3: [1, 6, 7],
    4: [1],
    5: [2],
    6: [3],
    7: [3]
}

# 방문한 노드를 기록할 리스트
visited = []

def dfs(cur_v):
    # 1. 현재 노드를 방문 리스트에 추가
    visited.append(cur_v)

    # 2. 인접한 node를 순회하며 방문
    for v in graph[cur_v]:
        # 3. 만약 아직 방문하지 않은 노드라면
        if v not in visited:
            # 재귀적으로 다음 노드 방문 처리
            dfs(v)

    return visited

# --- 테스트 실행 ---
print(f"방문 순서: {dfs(1)}")
# 출력 결과: [1, 2, 5, 3, 6, 7, 4]
